# Industrial-Analytics-Platform
Industrial analytics project focused on production monitoring, downtime analysis and process performance evaluation using SQL, Python and Power BI.
The idea of this project is to realize a simple way to examine and understand the processes going on in an example industry.

## Project objetives:
  _Production analysis
  _Downtime causes identifying
  _Efficiency measuring
  _Stability monitoring

## Technologies

- SQL
- Python
- Power BI

## Dataset

Manufacturing Downtime Dataset (Soda Bottling Production Line)

## KPIs

- Total Production
- Downtime Minutes
- Downtime by Cause
- Production Time
- Average Batch Duration
- Production by Product

## Notations
_Calculating time for each batch to be produced, an issue was discovered. Time resulted <0 when the batch finished after midnight. This issue was solved on the batch_performance view at the SQL query.
